Express + TypeScript学習用API

このAPIは、Express.js + TypeScriptでのバックエンド開発を学習するためのサンプルAPIです。

主な機能

  • ユーザー管理 (CRUD)
  • 投稿管理 (CRUD)
  • 認証・認可
  • バリデーション

認証方法

現在は認証なしでアクセス可能ですが、将来的にJWT認証を実装予定です。

レート制限

  • 15分間に100リクエストまで
  • 制限を超えると429エラーを返します

Users

ユーザー管理API

ユーザー一覧取得

登録されているユーザーの一覧を取得します

query Parameters
page
integer >= 1
Default: 1

ページ番号

limit
integer [ 1 .. 100 ]
Default: 10

1ページあたりの件数

Responses

Development server

http://localhost:3000/api/v1/users
